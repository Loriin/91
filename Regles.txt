Nombre de joueur : 2


Le plateau :

- 3 terrain : fields (alignement) bon, mauvais, neutre : Chaque field possede 7 places

- score:

- pile de buff / debuff => Buff dépend de la compinaison et de l'alignement du field

- Une main (contien de 0 à 1 carte)


Commun : 

- 4 terrain commun sans Alignement qui appartiene au deux joueur => possede 5 place (20 cartes)

- quand tu remplis un terrain commun => il ce vide ENTIEREMENT dans la pioche


Un tour :

2 phase : 

DrawPhase:
	- piocher dans un de field
	- piocher sur la pioche
	- piocher dans un terrain commun
MainPhase:
	- Pose sur un de ces field
	- pose sur un terrain commun

EndPhase:
	- tour de l'adversaire

ServerPhase:
	- Attente du serveur:

Les carte :

1 * 5 roi		8	10
1 * 5 reine		7	9
2 * 5 chevalier		5	6
2 * 5 clerc		4	6
3 * 5 valet		2	3
9 * 5 noble		1	2
1 mage			30



Les commandes : 


sf::Uint32 ou sf::Int32

trame de commande : [numéro de commande][advs][arg1][arg2][arg3][arg4] chaque case est un Uint32 (or advs)

advs = true => commande pour advs
advs = false => cmd pour soit


0 - 9 -> changement phase

[0][advs] (pas d'argument) = DrawPhase
[1][advs] = MainPhase
[2][advs] = EndPhase
[3][advs] = ServerPhase
[4][advs] = Erreur

10 - 99 -> action de jeu

[10][advs][arg1][arg2][arg3][arg4] = joueur 'advs' pioche une carte. arg1 = TYPE. arg2 = FACTION, arg3 = valeur simple, arg4 = valeur de groupe
[11][advs][arg1][arg2] = joueur advs pioche une carte sur le fiel numéro arg1 à l'emplacement arg2
[12][advs][arg1][arg2] = joueur advs pioche une carte sur le terrain commun numéro arg1 à l'emplacement arg2

[13][advs][arg1][arg2] = joueur advs pose une carte sur le fiel numéro arg1 à l'emplacement arg2
[14][advs][arg1][arg2] = joueur advs pose une carte sur le terrain commun numéro arg1 à l'emplacement arg2

100 - 999 -> action de terrain et environement

[100][advs][arg1] = le joueur advs vois son fiel numéro arg1 détruit (tous va dans la pioche)
[101][advs][arg1] = le joueur advs vois le terrain commun numéro arg1 détruit (tous va dans la pioche)
[102][advs][arg1] = le joueur advs gagne
[103][advs][arg1] = le joueur advs perd

[105][advs][arg1] = le score du joueur advs est arg1
